CSS Box Model

The CSS Box Model is a structure which includes a few properties which help segment and design how a website is layed out. These elements are the content of the website, the padding of the content, the margin of the padding and finally the borders of the margins. The box model resembles how these elements are in relation to one another, in the form of rectangular boxes.

Figure 1 - My interpretation of the CSS Box model, based on that from GeeksForGeeks,2018.

Each element of the CSS box model has a purpose in web design, and I will now go over the significance of each of them.

Content - The content of the CSS Box model is any detail on the website, such as images, text, or any other media that the client is using the web page to view.

Padding - Padding in a HTML file surrounds the content of of the file, but is still within the border of the website. Like other components, the height and width can be edited. The padding is often used to expand the size of the content, without editing the scale of the content.

Margin - The Margin section consists of the area between the border, and the margin, which is on the outer-most part of the diagram. The margin is used to position content in relation to the edges of the tab the web page is open in. The margin can be utilized to ensure all essential content is on the screen at once, such as a video player.

Border - The border is inbetween the margin section and the padding section of a HTML file. It is mostly used for wrapping the padding and content together, mostly for decorative purposes.

CSS Selectors

CSS selections are used within CSS to highlight a certain portion of the HTML sheet and change how it looks, often done by selecting a certain type of class or element. However, there are many different selections which can let the designer set elements to selected depending on an event. These are split into 5 different types of selectors, the first being simple selectors.

Simple selectors - Simple selectors get their name from the fact that they operate simply by selecting what the element or class is, for example, a paragraph or header. For example, the "Cascading Style Sheets" title of this page uses a simple CSS selector to make the element animated.

Figure 2 - Example of a simple selector.

Combinator selectors - Combinator selectors function slightly differently from simple selectors as they select elements depending on how they relate to each other. An example of a combinator selector would be selecting all elements which are inside another certain element, or elements that follow another element. I have used a combinator selector to make all of the galleries within this section have red text.

Figure 3 - Example of a combinator selector.

Pseudo-Class selectors - Pseudo-Class selectors refer to a state of a class which is different to the usual. For example, This could be when a mouse is hovered over a specific element or whether a link has been visited or not. In this website, I have used Pseudo-Class selectors for multiple elements and states, but the best example is that of the box which expands when you hover over it.

Figure 4 - Example of a pseudo-class selector.

Pseudo-Elements selectors - Pseudo-Elements allow you to alter a specific part of an elements within a HTML file. This can include certain segments of text, or content to precede or follow text, or other content of an element. This can be useful to bring attention to a specific reference within an element, or simply stylise a page better. For example, a Pseudo-Element selector has been used to change the first letter of each paragraph to a different colour and a larger size.

Figure 5 - Example of a pseudo-element selector.

Attribute selectors - Attribute selectors are quick useful as they allow the designer to select all of the elements in a HTML document with the same properties. This is useful for both stylisation and quickly changing a lot of different elements. In this website, I have used an attribute selector to make every other selector explanation in this segment change font and colour for stylisation when they are hovered over, which uses both pseudo-class and attribute selectors.

Figure 6 - Example of an attribute selector.
