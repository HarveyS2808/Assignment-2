CSS Box Model

The CSS Box Model is a structure which includes a few properties which help segment and design how a website is layed out. These elements are the content of the website, the padding of the content, the margin of the padding and finally the borders of the margins. The box model resembles how these elements are in relation to one another, in the form of rectangular boxes. (https://www.geeksforgeeks.org/css-box-model/)

Figure 1 - My interpretation of the CSS Box model, based on that from GeeksForGeeks,2018.

Each element of the CSS box model has a purpose in web design, and I will now go over the significance of each of them.

Content - The content of the CSS Box model is any detail on the website, such as images, text, or anything that the client is using the web page to view.

Padding - Padding in a HTML file surrounds the content of of the file, but is still within the border of the website. Like other components, the height and width can be edited.

Margin - The Margin section consists of the area between the border, and the margin, which is on the outer-most part of the diagram.

Border - Inbetween the margin section and the padding section of a HTML file.
