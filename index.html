CSS Box Model

The CSS Box Model is a structure which includes a few properties which help segment and design how a website is layed out. These elements are the content of the website, the padding of the content, the margin of the padding and finally the borders of the margins. The box model resembles how these elements are in relation to one another, in the form of rectangular boxes.

Figure 1 - My interpretation of the CSS Box model, based on that from GeeksForGeeks,2018.

Each element of the CSS box model has a purpose in web design, and I will now go over the significance of each of them.

Content - The content of the CSS Box model is any detail on the website, such as images, text, or any other media that the client is using the web page to view.

Padding - Padding in a HTML file surrounds the content of of the file, but is still within the border of the website. Like other components, the height and width can be edited. The padding is often used to expand the size of the content, without editing the scale of the content.

Margin - The Margin section consists of the area between the border, and the margin, which is on the outer-most part of the diagram. The margin is used to position content in relation to the edges of the tab the web page is open in. The margin can be utilized to ensure all essential content is on the screen at once, such as a video player.

Border - The border is inbetween the margin section and the padding section of a HTML file. It is mostly used for wrapping the padding and content together, mostly for decorative purposes.
