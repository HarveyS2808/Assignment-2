CSS Box Model

The CSS Box Model is a structure which includes a few properties which help segment and design how a website is layed out. These elements are the content of the website, the padding of the content, the margin of the padding and finally the borders of the margins. The box model resembles how these elements are in relation to one another, in the form of rectangular boxes. (https://www.geeksforgeeks.org/css-box-model/)
