External CSS

Figure 1 - External CSS file contents
Figure 2 - External CSS file being linked within header

External CSS involves writing characteristics for certain objects in another file, specifically made for CSS styling. This file is then linked to HTML documents within the header tag which get their object design from the CSS File. External CSS is particularly useful when a website with many HTML pages wants to use the same stylization across multiple pages.

In figure 1, you can see where I have used an external CSS file to change the characteristics of the navigation bar, the footer, and the body of the HTML files within this website. This is convienient for keeping the website professional, and also saving a lot of time and space by only using the code once, then accessing it from different files.
